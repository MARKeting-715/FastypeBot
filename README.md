# FastypeBot

FastypeBot — Telegram-бот для тренировки слепой печати с генерацией фраз через AI.

## Быстрый старт

1. Склонируйте репозиторий.
   ```bash
   git clone https://github.com/MARKeting-715/FastypeBot.git
   cd FastypeBot
   ```

2. Установите зависимости.
   ```bash
   pip install -r requirements.txt
   ```

3. Укажите секреты.
   Можно использовать переменные окружения:
   - `BOT_TOKEN` или `API_KEY` для Telegram Bot API
   - `OPENROUTER_API_KEY` или `AI_KEY` для OpenRouter

   При желании можно оставить старый формат с `config.py`, если он удобнее.

4. Запустите бота.
   ```bash
   python fastype.py
   ```

## Как это работает

- Бот показывает одно предложение на русском языке.
- После ввода он подсвечивает ошибки, считает скорость и точность.
- Можно менять модель прямо из меню.

## Поддерживаемые модели

- `gemini`
- `llama`
- `deepseek`
- `mistral`
- `qwen`

## Полезные заметки

- Если модель не отвечает, бот покажет понятное сообщение и предложит выбрать другую.
- Выбранная модель сохраняется между раундами.
- Секреты не нужно коммитить в репозиторий.
